# app_tweets

це React-проект для відображення карток користувачів з можливістю підписки на них.

Компонент Card відображає картку користувача з кнопкою Follow і лічильником фоловерів. Початкові значення стану компонента: флаг follow, що вказує на те, чи підписаний користувач на даного користувача, та лічильник фоловерів.

При клікові на кнопку Follow змінюється текст на кнопці та колір. Додатково до кількості фоловерів додається 1. При повторному клікові на кнопку, текст та колір змінюються на початкове значення, а кількість фоловерів зменшується на 1.

Стан компонента можна зберігати у localStorage, щоб при оновленні сторінки дані про підписку та кількість фоловерів не зникли.
